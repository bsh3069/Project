# Support Vector Machine
* 서포트 벡터 머신은 머신 러닝 분야 중 하나로 패턴 인식 및 자료 분석을 위한 지도 학습 모델이며, 주로 분류와 회귀 분석을 위해 사용된다.
* SVM 알고링즘은 주어진 데이터 집합을 바탕으로 하여 새로운 데이터가 어느 카테고리에 속할지 판단하는 선형 분류 모델을 만든다.
